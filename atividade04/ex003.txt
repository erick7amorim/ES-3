class Veiculo {
    void mover() {
        System.out.println("Movendo-se de forma genérica");
    }
}

class Carro extends Veiculo {
    // Carros herdam o método mover(), mas podem ter um comportamento específico
}

A classe Veiculo possui um método mover(), e a classe Carro herda esse método. No entanto nem todos os veículos 
compartilham exatamente o mesmo comportamento ao se mover.
